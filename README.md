图书管理系统(HTML+Servlet)
===
项目为期末实训项目，制作一个图书管理系统,设置学生和管理员两种账户<br>
前端通过在HTML页面中引用Vue、Element-ui进行页面的设计和交互、使用Jquery发送请求、使用Echarts显示借阅量最高的几位用户和借阅量最高的图书<br>
后端使用Servlet(只学了Servlet)
